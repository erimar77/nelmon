This is the first release of Nelmon which contains plugins for Nagios.

In this first release there are two scripts for Windows, which you run
through NSclient++.

Check Windows Update - Monitors if you have missing Windows updates
Check Certificate Expiration - Monitors your certificates and warns before expiration

For more information about Nelkit and how you use the tools. Please visit:
http://networklore.com/nelkit/
